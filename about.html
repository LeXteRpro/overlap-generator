Overlap Generator Notes

Notes

This is a Trial version for those who need a solution now. There is a full version under development that will have the full list of feature. Here are some limitations with the current version:

  • x2 Text Overlap pixel limit will vary from around 215 to 250px depending on the characters used. Be aware. This will be fixed in the future.
  • X2 Unit Overlap max pixel limit is 75.
  • X2 Briefing Text max is 129 pixels. However, depending on the characters the limit could be lower or higher. Again, be wary until we improve the formula.
  • Do not enter color codes (ex: <04>) into the input. Add your custom colors after the text is generated.
  • Tested with Scmdraft 2 0.8.0
  • Use the string editor

Overlap Methods to be implemented:

  • Display Text x2
  • Display Text x3
  • Unit Text x2

  • Briefing Text x2

This is experimentation. Feedback will help better our future design.

Starcraft Broodwar: Color List

--- NON-COLORS (USE ANYWHERE) ---

Values Description
<00> Null, automatically changes to <01>
<20> Space
<09> Tab
<12> Right Align / Overlap Back
<13> Center Align / Overlap Front
<14> Invisible (Overrides subsequent colors until new line)
<0A> New Line
<0B> Invisible (Overrides subsequent colors until new line)
<0C> Nothing?/New Line?
<1A> Nothing?/Periwinkle (Default color)
<7F> Filled square (in-game mission objectives), empty thin rectangle (mission briefing), lowered O (unit names). Can be colored.

--- IN GAME ---

Values Description
<01> Mimicks the second-last used color
<02> Periwinkle (Default color)
<03> Yellow (Detector)
<04> White (Advisor)
<05> Grey (Overrides subsequent colors until new line)
<06> Red (Disabled)
<07> Green (Blind/Parasite)
<08> Red (Player 1)
<0E> Blue (Player 2)
<0F> Teal (Player 3)
<10> Purple (Player 4)
<11> Orange (Player 5)
<15> Brown (Player 6)
<16> White (Player 7)
<17> Yellow (Player 8)
<18> Green (Player 9)
<19> Lemon (Player 10)
<1B> Peach (Player 11)
<1C> Royal Blue (Player 12)
<1D> Asparagus (Player 13)
<1E> Blue-Grey (Player 14)
<1F> Cyan (Player 15)

--- BRIEFING ROOM ---

Values Description
<01> Mimicks the second-last used color
<02> Periwinkle (Default color)
<03> Dark Green
<04> Light Green
<05> Grey (Overrides all subsequent colors on the string.)
<06> White
<07> Red

(Using other colors yields unpredictable results.)
